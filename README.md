## Endpoint Documentation
### Register User
#### Request
Method: `POST`  
Route: `/register`  
Request Body:  
```
{
    fullname: "Fubuki Shirakami",
    username: "fubuki",
    email: "fubuki@gmail.com",
    password: "Password123!"
}
```
#### Response
Status Code: `200`  
Response Body:
```
{
    data: {
        // user data
    },
    meta: {
        access_token: "access token",
        refresh_token: "refresh token",
        expires_at: 1000000,
    }
}
    
```

### Get Profile
#### Request
Method: `GET`  
Route: `/users/:user_id`  
Query Params: 
```
{
    filter: "tweets",// valid values are tweets, tweets_n_replies, media, likes  
}
```
Request Header:
```
{
    Authorization: "Bearer accesstoken",
}
```
#### Response
Status Code: `200`  
Response Body:
```
{
    user: {
        // user data
    },
    content_filter: "tweets", // values are tweets, tweets_n_replies, media, likes,
    content: {
        data: [
        // data of tweets, tweets and replies, media, or likes
        ],
        meta: {
            per_page: 20,
            next: "next page url",
            prev: "previous page url"
        }
    }
}
```
